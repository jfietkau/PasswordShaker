Permissions - PasswordShaker

In order to display the PasswordShaker icon only on pages that contain a password field, PasswordShaker needs permission to access the contents of all pages that you visit.

If you do not wish to grant this permission, you can simply close this tab. Without the extended permission, PasswordShaker will behave the same as if you had set the page icon to appear only when the master password is required.
